# xtracer

tracer library for yii2, compatible with jaeger.

## Usage

`$ vi config/web.php`

> 'on beforeAction' => ['XTracer\Tracer','beforeAction'],
> 'on afterAction' => ['XTracer\Tracer','afterAction'],
